Asynchronous Code

1. Synchronous code:
Synchronous code runs line by line. One line must finish before the next one starts.
Asynchronous code:
Asynchronous code does't block the program.
~ Some tasks take time (like fetching data, waiting for a timer, reading files).
~ Instead of freezing the program, JS continues running other code and comes back when the task is done.
Handles with: Callbacks, Promises, Async/await.
